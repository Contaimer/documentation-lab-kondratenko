# documentation-lab-kondratenko
Сервіс для оренди велосипедів - це система для оренди велосипедів і вирішує проблему низького рівня активного відпочинку, в нас можна орендувати велосипед для швидкого подорожування, або для прогулянки якщо не має свого, цей продукт для тих хто тільки хоче купити велосипед та він не впевнений що це сподобається чи буде багато часу на його використання.
